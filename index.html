How it works

  1. Import VR Mixin
    @import 'mixins/vr/vr-main';
  2. Set base typograhy
    $font-display: Arial, sans-serif;
    $cap-font-display: 0.02825;
    $px-font-size: 16;
    $px-line-height: 24;
    $modular-scale: ‘golden section’;
  3. Personal styling
    div { margin-bottom: $distance-s; }
    h1 { @include vr($font-size-m); }
    code { padding-bottom: $paragraph-trailer; }
    img { height: $size-m; }
