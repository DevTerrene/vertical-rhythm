How it works

  1. Base typograhy

    $px-font-size: 16;
    $px-line-height: 24;

    Converted to REM’s automatically.

  2. Modular scale

    $modular-scale: ‘golden section’;

    Diminished fifth, perfect fifth, minor sixth, golden section, major sixth, minor seventh…

  3. Vertical Rhythm

    h1 { @include vr($font-size-xl); }

    Standard range from extra small to extra large.

  4. Alignment

    div { margin: $distance-s; }
    img { height: $size-m; }

    Distances and sizes are based on the vertical Rhythm.
